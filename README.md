# 🏠 EliteHomes – House Rent Prediction & Property Management System

## 📌 Project Description

EliteHomes is an intelligent House Rent Prediction and Property Management System developed to simplify the property rental process for both tenants and property owners. The system combines Machine Learning with modern web technologies to provide accurate rent estimation, property management, booking services, and user interaction features through a centralized platform.

The application enables users to predict monthly house rent based on multiple property attributes such as location, size, furnishing status, floor information, and tenant preferences. Along with rent prediction, the platform provides property listing, booking management, review management, and dashboard analytics functionalities.

---

## 🎯 Objectives

* Predict house rent using Machine Learning algorithms.
* Provide an easy-to-use property management platform.
* Simplify property discovery and booking processes.
* Maintain booking and prediction history.
* Generate insights through dashboard analytics.
* Improve transparency in rental decision-making.

## 🔄 System Flow

1. User accesses EliteHomes.
2. User registers or logs in.
3. User selects Rent Prediction or Property Booking.
4. Property details are submitted.
5. Flask backend processes the request.
6. Machine Learning model predicts rent.
7. Results are displayed to the user.
8. Prediction history and booking records are stored in the database.
